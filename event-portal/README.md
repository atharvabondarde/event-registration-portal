# Event Registration & Management Portal

A premium SaaS platform for creating events, managing registrations, participants, schedules, notifications, and analytics. Inspired by Apple Vision Pro design (glassmorphism, floating panels, micro‑interactions).

## Tech Stack
- **Frontend**: React + TypeScript, Vite, Tailwind CSS, Framer Motion, Lucide React, Chart.js, react-big-calendar
- **Backend**: Node + Express, TypeScript, Prisma ORM, MySQL, Passport‑Google‑OAuth20, express‑session
- **Database**: MySQL (empty on init)
- **Auth**: Google OAuth 2.0 (primary)

## Design System
- Glassmorphism utilities (`bg-glass`, `backdrop-blur`)
- Dark background (Deep Space Blue, Midnight Black)
- Accent colors: Electric Blue, Indigo, Soft Violet, Ice White
